# Arduino Earthquake Detector

### What I did use?
- Arduino Nano (Personalized and Redesigned) and Arduino UNO
- ADXL335 3 Axis Accelerometer
- 2x16 LCD Display
- Led (Red and Green)
- Resistors
- Buzzer
- Jumper Cables
- Arduino IDE
- Altium Designer
- Processing 3
- Tinkercad
